## Database Requirements

The following tables must exist in the database:

1. `users` - User accounts
2. `hotels` - Hotel records
3. `subscription_plans` - Subscription plan definitions (with SERIAL id)
4. `payments` - Payment records
5. `subscriptions` - Hotel subscription records

**Note:** The `tables` and `dishes` tables are referenced in some queries but are not required for basic functionality.
